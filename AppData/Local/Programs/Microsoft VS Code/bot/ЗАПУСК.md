## 📋 ЧЕКЛИСТ ПЕРЕД ЗАПУСКОМ

### 1. ⚠️ Срочно: Создайте таблицы в Supabase

Без этого бот не будет работать!

**Быстрый способ**:
1. Откройте [SQL Editor](https://dgxzkcybwwonqqkisltl.supabase.co/project/_sql)
2. Скопируйте код ниже
3. Вставьте в редактор
4. Нажмите **Run**

```sql
-- Таблица leads
CREATE TABLE IF NOT EXISTS leads (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  telegram_id BIGINT NOT NULL UNIQUE,
  username TEXT,
  full_name TEXT,
  phone TEXT,
  interest TEXT,
  notes TEXT,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

CREATE INDEX IF NOT EXISTS idx_leads_telegram_id ON leads(telegram_id);
CREATE INDEX IF NOT EXISTS idx_leads_interest ON leads(interest);

-- Таблица messages
CREATE TABLE IF NOT EXISTS messages (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  lead_id UUID REFERENCES leads(id) ON DELETE CASCADE,
  role TEXT NOT NULL CHECK (role IN ('user', 'assistant')),
  content TEXT NOT NULL,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

CREATE INDEX IF NOT EXISTS idx_messages_lead_id ON messages(lead_id);
CREATE INDEX IF NOT EXISTS idx_messages_created_at ON messages(created_at);

-- RLS
ALTER TABLE leads ENABLE ROW LEVEL SECURITY;
ALTER TABLE messages ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Service role full access" ON leads FOR ALL TO service_role USING (true) WITH CHECK (true);
CREATE POLICY "Service role full access" ON messages FOR ALL TO service_role USING (true) WITH CHECK (true);
```

**Проверка**:
```sql
SELECT COUNT(*) FROM leads;  -- Должно вернуть 0
SELECT COUNT(*) FROM messages;  -- Должно вернуть 0
```

## 🛑 Если что-то не работает

### Ошибка "BOT_TOKEN не найден"
Проверьте `.env`:
```bash
BOT_TOKEN=7824488734:AAF-rjoXJAusc4_vgzbhOSa2OqppkHgZqCc
```

### Ошибка "SUPABASE_KEY должен быть задан"
Проверьте `.env`:
```bash
SUPABASE_SERVICE_KEY=...
```

### Ошибка "relation does not exist"
Таблицы не созданы! Выполните SQL скрипт из шага 1.

### Бот не отвечает
1. Проверьте что бот запущен (`python main.py`)
2. Проверьте интернет
3. Проверьте логи: `cat logs/bot.log`
