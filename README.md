# Парсер объявлений Авито для сравнительного анализа цен

Этот скрипт позволяет собирать данные о ценах объявлений с сайта Авито для проведения сравнительного маркетингового анализа.

## Требования

- Python 3.7+
- Библиотеки из requirements.txt

## Установка

1. Клонируйте репозиторий или скачайте файлы
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Использование

1. Запустите скрипт:
   ```bash
   python avito_price_parser.py
   ```

2. Введите URL поиска на Авито (например, `https://www.avito.ru/moskva/kvartiry` для поиска квартир в Москве)

3. Укажите количество страниц для парсинга (по умолчанию 1)

4. Скрипт соберет данные и сохранит их в файл `avito_prices.csv`

5. Будет выполнен базовый анализ цен:
   - Средняя цена
   - Медианная цена
   - Минимальная и максимальная цена
   - Стандартное отклонение

## Особенности

- Скрипт использует заголовки User-Agent для имитации браузера
- Между запросами делается пауза в 1 секунду чтобы избежать блокировки
- Поддерживает парсинг нескольких страниц
- Сохраняет результаты в CSV формате для дальнейшего анализа в Excel или других инструментах

## Важные замечания

- Парсинг сайтов может нарушать условия использования сервиса. Используйте этот скрипт ответственно и только для законных целей.
- Авито может менять структуру своей HTML-разметки, поэтому селекторы в скрипте могут потребовать обновления.
- Рекомендуется добавить дополнительные задержки и использовать прокси при большом объеме запросов.

## Результаты

После выполнения скрипта вы получите:
- Файл `avito_prices.csv` с собранными данными
- Консольный вывод с анализом цен
- Список первых 5 объявлений для быстрой проверки

Эти данные можно использовать для:
- Сравнения цен на аналогичные товары/услуги
- Определения среднерыночной стоимости
- Выявления аномалий в ценообразовании
- Анализа конкурентной среды
